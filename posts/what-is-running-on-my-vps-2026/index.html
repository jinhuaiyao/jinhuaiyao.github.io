2 个月前重装了 VPS 的系统，利用 AI 实践了一把服务器上所有数据和服务的备份和恢复。这次让 AI 直接 ssh 到服务器上进行检查和安全加固。

2026 年 7 月更新一下云服务器上最新的应用。

Nginx

Cloudflare → nginx → 本地后端。nginx 负责反向代理和 TLS，后端服务全部只绑在 127.0.0.1 上，不直接暴露在公网。

除了反代自己的应用，还做了三个 API 中转，在国内不用科学途径也能直接调用。

OpenAI：https://x1.xxx.com/v1 → api.openai.com

Notion：https://x2.xxx.com/v1 → api.notion.com

Gemini：https://x3.xxx.com → generativelanguage.googleapis.com

docker 里运行以下应用。

code-server

在浏览器里访问 VS Code，任何机器上打开网页就能写代码、管理代码片段。

Planka

看板应用，类似 Trello，工作上的 task 都放在这里，数据在自己手上。

Memos

轻量笔记服务，我把它用在博客里，调用 memo s的 api 展示一些只言片语。

Bark

iOS 推送服务，监控脚本、定期提醒都靠它给 iPhone 推消息。

doocs/md

公众号排版工具。

自己定制的应用

这一年多和 AI 一起写了几个小应用。

错题本

给小朋友整理错题用的。花一小时，我和 AI做了个错题本 app

复习卡片

给小朋友用的学习卡片工具，这个目前没怎么用。

小朋友的进步手册

记录小朋友的日常进步。当他遇到困难、哭鼻子的时候就打开给他看看，让他回顾一下通过努力得到的进步。

微信文章存 Notion

看到好的微信文章，复制链接，轻点 iPhone 背面三下，调用快捷指令，就可以存到 Notion 里归档。

微信读书划线卡片

把微信读书的划线做成随机回顾卡片

安全

ufw 防火墙：只放行 SSH 和 Cloudflare 网段的 80/443。

fail2ban：防 SSH 暴力破解。

sshd：非标准端口，仅密钥登录。
