备份备份再备份

2022/02/24

昨天有个朋友家用 NAS 升级系统后资料不能访问了，数据没有其他备份，很难办。

这里再次强调一下重要资料备份的重要性，不要等到数据丢失了才追悔莫及。

看这篇旧文章，好好珍惜自己的资料。

目前我的备份设备是家用 4 盘位 NAS，硬盘 1 和 2 是去年年底新更换的 4T 硬盘，硬盘 3 和 4 是 2015 年买的。NAS 外挂一块 2T 移动硬盘做为备份盘。

备份策略如下：

  • 工作笔记 -> OneDrive （直接存放）
  • 工作笔记 -> iCloud Drive （via GoodSync macOS app ）
  • 手机相册 -> OneDrive （via OneDrive iOS app ）
  • 手机相册 -> NAS硬盘 1 -> 外置移动硬盘 （via QNAP iOS app, QNAP internal app）
  • DEVONthink -> NAS硬盘 1 -> 外置移动硬盘 （via 每月手动导出, QNAP internal app）
  • DEVONthink -> 家用 mac -> Time machine （via 自带同步, 每月手动开启 Time machine）
  • DayOne -> NAS硬盘 1 -> 外置移动硬盘 （via 每月手动导出, QNAP internal app ）
  • 其他重要资料 -> OneDrive -> NAS 硬盘 2 -> NAS 硬盘 4 -> 外置移动硬盘 （via QNAP internal app）

希望大家能保护好自己的重要资料。
