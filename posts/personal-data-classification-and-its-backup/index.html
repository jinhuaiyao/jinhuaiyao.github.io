个人资料分级及备份

2023/12/06

“自己产生的，独有的数据，才是值得备份的。网上下载来的，往往没那么重要。” —— ze ran

以前写过一些:

好好珍惜自己的资料
备份备份再备份

更新一下 2023 版，目前所有重要资料的备份都是自动进行的，不再有手动操作。

个人照片和视频

存于 iCloud， 下载并保留原片到 Mac mini 的外接移动硬盘，Mac mini 及移动硬盘通过 Time Machine 备份，Time Machine 配置两块移动硬盘，每小时轮流备份。

个人文档资料

  • 日记
  • 博客文章
  • 个人 Wiki（记录和总结，阅读笔记）
  • 曾经的工作笔记
  • 账号和密码
  • 其他个人资料（例如投资统计，体检报告，证件电子版，备份文件）

日记，博客文章，个人 Wiki，曾经的工作笔记，存于 Dropbox，通过 Mac mini 的 Time Machine 备份。

账号和密码存于 1Password，1Password 主密码有备份，其他账号密码都交给 1Password 了。

其他个人资料主要存于 DEVONthink，通过 iCloud 同步到 Mac mini，再通过 Time Machine 进行备份。部分大文件存于 OneDrive，通过 Mac mini 的 Time Machine 备份。

其他的收集

  • 收集的软件和工具
  • 收集的学习资料（英语，数据库，编程；文档，图书，音频，视频）
  • 收集的文档、图书
  • 收集的文章及标注
  • 收集的剧集

其中相对重要的学习资料会存于 DEVONthink，通过 Time Machine 备份。不重要的收集（比如剧集）存于百度盘，NAS 里也会放一份，读取速度相对比较快。
