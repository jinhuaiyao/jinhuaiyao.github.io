UPDATE - M1 芯片 MacBook Pro 上搭建 Oracle 实验环境

2023/10/22

距离 M1 芯片 MacBook Pro 上搭建 Oracle 实验环境已经过去了一年。

今年 6 月底 Oracle 宣布支持 Arm 架构上 Oracle 数据库。7 月 9 号我在自己的 M1 芯片的 MacBook 上虚拟机里成功安装了 Oracle 数据库，补记一下。

Oracle Linux

下载、安装 Oracle Linux 8.8 Arm version

参考 Database Installation Guide for Linux

https://yum.oracle.com/oracle-linux-isos.html，下载OracleLinux-R8-U8-Server-aarch64-dvd.iso

在 VMware Fusion 里安装即可。

Oracle Database 19c

下载 Oracle Database 19c (19.19) for LINUX ARM (aarch64)

https://www.oracle.com/database/technologies/oracle19c-linux-arm64-downloads.html

安装 Oracle 19c 数据库

安装依赖包 dnf -y install oracle-database-preinstall-19c.aarch64

然后到解压后的 database 目录里运行 ./runInstaller，图形化安装即可。
