创造从模仿开始

 2022-01-30  460 words   1 minute 

最近除了学习象棋就是学习 Python，象棋不谈也罢，Python的学习是通过学习 flask 框架进行的。

这里推荐一下 https://blog.miguelgrinberg.com/post/the-flask-mega-tutorial-part-i-hello-world。

可以按照教程一步一步从无到有、从简单到复杂地搭建一个网站并且实现很多功能。

可以一行一行照着教程敲代码，熟悉 Python 的使用方法，同时训练自己的专注能力（代码很容易就会有缺漏，运行的时候就会获得一堆报错，笑）。

可以在某些地方做一些更改，看看更改之后有什么效果。

可以立即看到页面效果，不会简单枯燥地学习 Python 语法，不断点击页面做测试也很有成就感。

可以学以致用，模仿、熟悉框架以后可以搭建一个自己的网站，发挥自己的创造力。

可以再学习一些 CSS 和 Javascript，让网站更生动。

学习过程中会遇到很多一时难以理解的地方，也会有些报错需要排查，可以借此来挑战一下自己。

学到第10章就已经有了许多网站一样的功能。

有登录登出

有发贴

有关注取关

有编辑个人信息

有密码重置


大家可以尝试一下。
